CP1406/CP1806/CP5638 - Assignment 1 - Project Plan

Name: Petite Treats Website

Customer Mission

Petite Treats aims to improve their business operations by increasing their revenue through the creation of a new website that will promote their products on the web, increase their online presence and reach their target audience more effectively.

Goals

  • Create a website to increase revenue for Petite Treats
  • The new website promotes the products offered by the company using product images
  • The new website encourages new and repeat customers with its user-friendly interface
  • The new website allows customers to submit orders and enquiries

Success Evaluation

  • Advertising for the new website will be consistent for a period of 3 months
  • Over the 3-month period, the number of orders submitted on the website will be counted and compared monthly to evaluate and assess the volume of orders generated on the website at the end of each month

Target Audience

  • Young families and young people
    • local residents
    • between 17 to 35 years old
    • students and working professionals
    • to cater to the behavior and personalities of the target audience, the website will be designed with a modern feel:
      • right selection of site colors the are bright and lively
      • typography will also play an imporant role in achieving a modern look and feel, thus a mix of cartoon-style fonts and sans-serif fonts will be used.
      • present a variety of images to cater to the younger audience who love graphics
      • a conversational tone for the website content will be used as opposed to a formal tone to create a friendlier atmoshpere for site visitors

Usability Test

  1. Test Plan

    The usability test begins with asking each user to perform the following test cases below. This also gives users a chance to explore the website to generate feedback:

    Test Case Number Goal Task Expected Result
    1 View all available products
    1. From homepage, click 'SHOP THE MENU' button on the hero banner
    2. Scroll through the menu page
    		 The user navigates to a new page and sees each available product in a grid layout.
    2 View product information
    1. Navigate to MENU page from the navigation bar
    2. Scroll through the page
    3. Select "View product" link on a product tile
    		 A product page opens with the product name, product price, product description and order now button displayed.
    3 Order a product
    1. Navigate to MENU page from the navigation bar
    2. Scroll through the page
    3. Select "View product" link on a product tile
    4. Navigate to product page of selected product
    5. Click on the "Order Now" button
    		 The user is navigated to the Order confirmation page for successful order request.
    4 Contact for inquiries
    1. Navigate to CONTACT page from the navigation bar
    2. Fill out contact form
    3. Submit contact form
    • Successful Submission: User receives a confirmation of successful submission.
    • Unsuccessful Submission: If user has missing input fields, an error prompt is shown.
    5 Join Newsletter
    1. Navigate to NEWSLETTER page from the navigation bar
    2. Fill out sign up form
    3. Submit sign up form
    • Successful Submission: User receives a confirmation of successful submission.
    • Unsuccessful Submission: If user has missing input fields, an error prompt is shown.
    6 View products with special deals
    1. Navigate to WEEKLY SPECIALS page from the navigation bar
    		 The user navigates to a new page and sees each available product in a grid layout.
    7 Learn more about the shop
    1. Navigate to ABOUT page from the navigation bar
    		 The user navigates to a new page and sees content about the store and its customers.
